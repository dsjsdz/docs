# 📦 订单管理

用于记录并展示售卖机产生的所有交易订单。商户可通过该模块实时查看销售情况、追踪订单状态、处理退款及异常出货问题。

订单数据由售卖机在完成交易后自动同步至云平台，包括支付信息、商品详情、出货状态等。

主要功能

1. 📋 订单列表

系统以时间顺序展示所有交易订单，可通过筛选或搜索快速定位目标订单。

支持的筛选条件包括：

订单编号

售卖机编号 / 名称

商品名称

支付方式（微信、支付宝、余额等）

订单状态（待出货、出货成功、出货失败、已退款等）

时间范围（下单时间 / 出货时间）

2. 🔍 订单详情

点击任意订单，可查看完整交易信息：

订单编号

售卖机编号 / 名称

商品名称与数量

订单金额与支付方式

用户信息（如支持会员系统）

出货状态与出货时间

交易流水号（第三方支付返回）

创建时间 / 更新时间

3. 🚚 出货状态追踪

系统会自动标记订单的出货流程，包括：

状态 说明
待出货 支付成功，等待机器执行出货命令
出货中 售卖机正在执行出货操作
出货成功 出货成功，交易完成
出货失败 设备反馈出货异常
已退款 对出货失败或取消的订单完成退款

⚠️ 平台会自动检测售卖机返回的出货结果，如设备离线或反馈异常，订单将标记为“出货失败”并可进入人工复核。

4. 💰 退款与异常处理

商户可针对异常订单进行退款或手动修正。

支持单笔退款（通过原支付渠道退回）

支持异常备注记录（如“货道卡货”“用户未取货”）

支持导出退款记录报表

5. 📈 数据统计与导出

订单模块支持多维度的销售统计和报表导出：

销售额、交易笔数趋势图

商品销量排行

售卖机销售排行榜

支付方式统计

导出为 Excel / CSV 报表

三、角色与权限
角色 权限说明
管理员 查看与操作所有商户订单
商户 查看与操作名下售卖机订单
区域管理员 仅可查看负责区域内设备订单
财务人员 可查看订单金额与退款记录
四、使用示例

例：商户 “张三便利” 登录后台 → 进入 订单管理 → 筛选“学校A区售卖机” → 查看昨日出货记录
系统显示：

总订单：128 笔

成功出货：125 笔

出货失败：3 笔（2笔已退款，1笔待处理）

销售额合计：¥842.50
