## 1. 货道详情

![merchant](/images/2025-11-23_144728.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

- 货道名称: 比如 A1
- 层号：0，从 0 开始
- 列号: 0，从 0 开始
- 货道容量: 7，单个货道的能放多少个商品
- 电机号: 0，从 0 开始，最大 99

## 2. 新建货道

白色按钮的货道是未分配的，可以新建货道

![merchant](/images/2025-11-23_145057.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

## 4. 货道容量

修改货道容量会自动更新关联的产品容量。

## 5. 删除货道

![merchant](/images/2025-11-23_145237.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

删除货道会自动将相关关联的商品删除。
