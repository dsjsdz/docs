# 🛒 商品管理

一、功能概述

商品管理模块用于商户对售卖机中销售的商品进行全面管理，包括商品的上架、下架、库存维护、价格设置、分类管理以及促销活动管理。
通过商品管理，商户可以高效维护多台售卖机的商品信息，实现统一调控和销售优化。

二、主要功能

1. 📋 商品列表

系统展示商户名下所有商品，可按条件筛选和搜索：

商品名称

商品编号 / SKU

所属售卖机

商品分类（饮料、零食、食品等）

上架状态（已上架 / 已下架 / 草稿）

库存数量

2. ➕ 新增商品

商户可以手动或批量新增商品：

填写商品基本信息：名称、图片、描述、单位、条形码

设置价格、折扣和活动价格

分配所属售卖机和货道

设置库存初始值

选择商品分类

3. ✏️ 编辑商品

对已存在商品进行修改，包括：

修改价格或折扣

调整库存数量

修改商品图片或描述

更改所属售卖机或货道

上架 / 下架操作

4. 📦 库存管理

查看各售卖机货道的库存数量

系统可实时同步售卖机库存状态

支持批量补货操作或库存调整

支持库存预警（低于设定阈值自动提醒商户补货）

5. 🎯 商品分类与分组

支持创建自定义分类（如饮料、零食、健康食品）

可将商品按分类或主题分组，方便售卖机上架管理

支持分类快速筛选和批量操作

6. 📈 销售与数据分析

查看每个商品的销售量、销售额、出货成功率

根据销售数据优化库存和补货策略

支持按时间范围导出报表（Excel / CSV）

7. 🔔 促销与活动管理（可选）

设置商品折扣、满减或限时活动

活动商品可同步到对应售卖机，自动生效

支持活动数据统计（销量、收益、库存消耗情况）

三、权限与角色
角色 权限说明
商户管理员 查看、添加、编辑、删除名下商品
区域管理员 查看负责区域售卖机商品，但不能修改
财务人员 查看商品销售数据与价格信息，不可操作库存或上架状态
技术支持 可查看商品绑定的设备信息和库存状态
四、使用示例

例：商户“张三便利”想在“学校A区售卖机”上架新零食

登录后台 → 商品管理 → 新增商品

填写商品信息：名称“巧克力棒”，分类“零食”，价格 ¥5，库存 100 件

分配货道 A01 → 保存 → 系统同步到售卖机

商品状态显示“已上架”，可直接销售
