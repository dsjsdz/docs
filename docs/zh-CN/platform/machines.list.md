# 🖥️ 设备管理

一、功能概述

设备管理模块用于商户或平台对售卖机设备进行统一管理，包括设备信息、状态监控、远程控制及维护记录。
通过设备管理，商户可实时掌握每台售卖机的运行情况，优化补货、维护和出货流程。

二、设备与商户关系

每台设备必须绑定一个商户

一个商户可以拥有多台售卖机设备

设备信息与商户、商品、订单、出货日志等模块关联

三、主要功能

1. 📋 设备列表

显示所有售卖机设备信息

支持筛选和搜索：

设备编号 / 名称

所属商户

设备型号

设备状态（在线 / 离线 / 故障）

所在位置或区域

注册时间

2. 🔧 新增 / 编辑设备

添加新设备：填写设备编号、型号、所属商户、安装位置

编辑设备信息：修改设备名称、位置、所属商户

绑定或解绑设备至商户

设置设备分组或区域分类

3. ⚡ 设备状态监控

实时查看设备在线状态

查看设备运行参数：

温度、湿度、功率、电量

库存状态

出货指令执行状态

设备异常报警提示：离线、出货失败、硬件故障等

4. 🔄 远程控制与维护

支持远程操作：

重启设备

更新广告或商品信息

同步库存

查看历史操作记录与设备维护日志

5. 📈 数据统计与报表

按设备维度统计：订单数量、销售额、出货成功率

异常事件统计：故障次数、离线时长

支持导出设备报表用于管理或售后维护

四、权限与角色
角色 权限说明
平台管理员 可管理和查看所有设备信息、状态和操作记录
商户管理员 查看和操作名下设备信息、状态和库存
区域管理员 查看负责区域的设备信息与状态
技术支持 查看设备状态、日志及维护记录，执行远程操作
五、使用示例

例：商户“张三便利”管理“学校A区售卖机”

登录后台 → 设备管理 → 搜索“学校A区售卖机”

查看设备状态：在线，温度 25°C，库存正常

发现设备 A05 出货失败 → 远程重启设备

系统记录操作日志 → 下次可追踪维护记录

查看销售统计 → 今日出货 120 笔，总销售额 ¥950
