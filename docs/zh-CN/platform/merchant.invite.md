# ➕ 2.2 邀请码

只能通过商户管理员创建邀请码，且12小时内只能创建一个邀请码，过期后重新创建即可。

![merchant-invite-b](/images/merchant-invite-b.png){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

点击邀请码按钮即可获取邀请码

![merchant-invite-c](/images/merchant-invite-c.png){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

⚠️ 邀请码12小时内有效，请及时使用。
