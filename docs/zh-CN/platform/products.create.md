🏷️ 产品管理
一、功能概述

产品管理模块用于商户或平台对销售商品的统一管理，包括创建产品模板、设置规格、组合商品以及管理商品生命周期。
该模块支持在多台售卖机间共享同一产品信息，提高管理效率和数据一致性。

二、主要功能

1. 📋 产品列表

显示商户所有已创建的产品

支持筛选与搜索：

产品名称

产品编号 / SKU

产品分类（饮料、零食、组合套餐等）

上架状态（已上架 / 草稿 / 停用）

2. ➕ 新建产品

商户可创建新产品模板：

填写产品基本信息（名称、图片、描述）

设置规格（容量、口味、包装类型等）

关联对应的单品或组合商品

设置价格、成本和库存策略

选择适用售卖机或货道

3. ✏️ 编辑产品

修改产品信息（名称、图片、描述）

调整规格或组合内容

更改价格、成本、上架状态

可批量更新已绑定售卖机的产品

4. 📦 产品库存管理

产品库存可通过单品库存自动计算

支持产品级别的库存预警

可导出库存数据用于分析或补货决策

5. 🎯 产品分类与标签

支持按类别（饮料、零食、组合套餐）管理产品

可给产品打标签（热销、新品、促销）

方便在售卖机上快速上架或推广

6. 📈 产品销售统计

统计各产品的销量、销售额、出货成功率

可按时间段、售卖机或货道维度分析

支持导出 Excel / CSV 报表

7. 🔄 产品生命周期管理

草稿 → 上架 → 停用 → 删除

支持批量上架或下架

保留历史销售数据，方便统计与复盘

三、权限与角色
角色 权限说明
商户管理员 查看、添加、编辑、删除产品模板，批量同步到售卖机
区域管理员 查看负责区域产品信息，不可修改
财务人员 查看产品价格和销售数据，不可操作库存
技术支持 可查看产品绑定设备和库存状态
四、使用示例

例：商户“张三便利”要上架新品“组合早餐套餐”

登录后台 → 产品管理 → 新建产品

填写信息：名称“组合早餐套餐”，包含“面包+牛奶+鸡蛋”

设置价格 ¥15，库存策略与单品同步

选择适用售卖机 → 保存 → 系统同步到售卖机

产品状态显示“已上架”，用户可直接购买
