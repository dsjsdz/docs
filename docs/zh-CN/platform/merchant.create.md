# ➕ 2.1 开通/加入商户

![merchant-create](/images/merchant-create.png){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

尚未绑定商户时，会默认进入引导商户开通页面，点击「开通商户」按钮进入开通商户流程，或者点击「加入商户」按钮进入加入商户流程。

## 2.1.1 开通商户

开通商户可对售卖机进行「商品上下架」操作

![merchant-form](/images/merchant-form.png){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

⚠️ 平台定期审核商户信息，防止异常账户或无效商户操作系统

| 字段名 | 是否必须 | 描述     |
| ------ | -------- | -------- |
| 名称   | ✅       | 商户名称 |
| 介绍   | ✅       | 商户介绍 |
| Logo   | -        | 商户头像 |

> 提交表单后，需要等待审核。
> 审核通过后会自动开通商户，且会自动更新用户角色与菜单、权限等相关信息。

## 2.1.2 加入商户

![merchant-invite](/images/merchant-invite.png){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

> [邀请码](/zh-CN/platform/merchant.invite) 12 小时内有效，请及时使用。

邀请码关联 `商户信息`，使用后可以选择捆绑商户，会自动加入到对应商户成员列表中。

![merchant-invite-code](/images/merchant-invite-code.png){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}
