# 1.1 语言切换

## A. 登录页面切换

![language](/images/language-a.png){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

> 在未登录状态下，切换后会自动切换语言。

## B. 登录后页面切换

![language](/images/language-b.png){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

> 在登录状态下，可以点击管理页面的 header 区域进行切换语言。

## C. 支持的语言

| 语种     | 版本支持      |
| -------- | ------------- |
| 中文简体 | v1.2.0 及以上 |
| 中文繁体 | v1.2.0 及以上 |
| 英语     | v1.2.0 及以上 |
| 西班牙语 | v1.2.0 及以上 |
| 葡萄牙语 | v1.2.4 及以上 |

> 其他语言目前不支持，如有语种需求可联系 <a href="mailto:cleveng@gmail.com">开发者</a>。
