# 1.1 注册/登录

官网提供多种登录方式，直达：[https://dash.awish.vip](https://dash.awish.vip/dashboard/login?redirect=/dashboard/profile)

![login](/images/login.png){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

| 平台      | 说明                                    | 注册            | 登录 |
| --------- | --------------------------------------- | --------------- | ---- |
| 微信扫码  | 国内外拥有微信 APP 的用户可使用微信扫码 | ✅              | ✅   |
| 谷歌授权  | 网页端用户可使用谷歌授权登录            | ✅              | ✅   |
| 邮箱&密码 | `邮箱 & 密码`，用户可自行更换           | ❌ 平台自动生成 | ✅   |

## A. 微信扫码（动态码）

![qrcode](/images/qrcode.jpeg){style="background: #fff; padding: 1rem; border-radius: 0.5rem; filter: blur(8px);"}

> 显示二维码后使用手机打开微信进行【扫一扫】，系统会自动解析扫码结果，如成功登录后会自动跳转到管理页面。

## B. 谷歌授权

直达: <a title='跳转授权页' href="https://dash.awish.vip/auth/login?app=gg&name=google" target="_blank">谷歌授权页</a>

> 点击谷歌授权会自动跳转到谷歌第三方授权页面，同意后系统会自动解析授权数据，未注册用户会自动进行注册，并成功登录。

## C. 邮箱&密码

![login-form](/images/login-form.png){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

> 通过平台提供的邮箱地址、登录密码即可
