# 1.1 用户注册

账户登录时间、客户端、IP地址

| 平台    | 说明                                   |
|-------|--------------------------------------|
| 微信扫码  | 会自动注册账号，若关联的 `open_id` 已经存在同一个账号，即登录 |
| 谷歌授权  | 根据谷歌授权获取用户信息并授权于平台                   |
| 邮箱&密码 | 使用平台提供的 `邮箱 & 密码` 进行登录               |

## A. 微信扫码

![qrcode.jpeg](/images/qrcode.jpeg){style="background: #fff; padding: 1rem; border-radius: 0.5rem"}

微信公众号

> 使用手机打开微信进行【扫一扫】，系统会自动解析扫码结果，如成功登录后会自动跳转到管理页面。

## B. 谷歌授权

> 点击谷歌授权会自动跳转到谷歌第三方授权页面，同意后系统会自动解析授权数据，未注册用户会自动进行注册，并成功登录。

## C. 邮箱&密码

> 通过平台提供的邮箱地址、登录密码即可
