# ️7.1 类型列表

允许商户自行创建售卖机类型设备类型，在创建设备时可以选择对应的设备类型。

![merchant](/images/2025-11-23_111141.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

售卖类型是根据实际的售卖机进行创建的，比如售卖机单个板卡支持 10 层 \* 10 列 货道类型电机，在实际生产的过程中，可能不会出现满货道的情况，因此配置设备类型进行关联。

## 1. 使用货道

![merchant](/images/2025-11-23_111612.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

如图所示，该设备类型仅有黑色按钮的货道是有效的，其他的按钮都是无效的；在绑定的商品中，无效的货道号在实物售卖机中也是不存在的。

## 2. 板卡选择

![merchant](/images/2025-11-23_112007.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

⚠️ 目前支持多种板卡类型，但是后续会根据实际情况进行扩展。

| 名称               | 描述              | 最大货道数 | 是否支持 |
| ------------------ | ----------------- | ---------- | -------- |
| Wish-noodle-v1.0   | 泡面机            | 10         |          |
| W50x-v1.0          | W50x 称重板       | 32         | ✅       |
| Wish-eLock24P-v1.0 | 24 货道电磁锁     | 24         | ✅       |
| Wish-M2301-v1.4    | 鼎商驱动板 M2301  | 100        | ✅       |
| Wish-ACH370-v1.0   | 鼎商驱动板 ACH370 | 100        |          |

`Wish-noodle-v1.0`, `Wish-ACH370-v1.0` 等系列板卡需要商户单独联系开发者。

手动添加 `machine_types` 商户支持的机器类型列表，否则无法选择。
