# 8.3.6 取货码

![2025-11-23_153307.jpg](/images/2025-11-24_090914.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

生成的取货码有效期为2小时，UTC时间。

## 1. 生成取货码

![2025-11-24_092335.jpg](/images/2025-11-24_092335.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

选择对应的货道，生成取货码

![2025-11-24_092335.jpg](/images/2025-11-24_092418.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

> 数量: 可选 1-3 个，即单次货道掉落的商品数量。

⚠️ 设备未开启购物车功能（有光眼检测的设备）只能设置1个，同一个货道只能有一个取货码，新添加的取货码会同步到APP设备上。

## 2. 删除取货码

![2025-11-24_092241.jpg](/images/2025-11-24_092241.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

⚠️ 取货码未使用自动过期，删除的取货码会同步到APP设备上。
