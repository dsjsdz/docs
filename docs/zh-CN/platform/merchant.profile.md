# ️ 2.3 商户信息

创建商户/加入商户都可以获取到商户信息

![merchant](/images/2025-11-22_160420.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

商户关联下的所有售货机可以分配给其他管理员。

## 编辑商户

![merchant](/images/2025-11-22_160609.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

已通过审核的商户可以自行修改 `商户名称`, `商户介绍`, `商户LOGO`。

> 商户一经审核通过，将无法删除，可申请更换管理员。
