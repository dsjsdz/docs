🍜 食谱配方管理（拉面机专用）
一、功能概述

食谱配方管理用于商户或管理员对拉面机的产品制作流程进行配置和管理，包括原料配比、工艺流程、出餐规格、价格及生产参数。
通过食谱配方管理，可保证每次出餐的口味一致性和制作效率，并方便批量更新或调整。

二、主要功能

1. 📋 食谱列表

展示所有拉面机可制作的食谱

支持筛选和搜索：

食谱名称（如“麻辣牛肉拉面”）

食谱编号 / SKU

分类（拉面、汤底、配菜等）

上架状态（启用 / 停用 / 草稿）

2. ➕ 新建食谱

填写基本信息：名称、描述、图片

配置食材原料：面条、汤底、配菜、调料等

设置原料数量、重量、配比比例

设置制作流程和工艺参数：煮面时间、汤温、搅拌顺序等

设置出餐规格：分量、口味强度、加料选项

设置价格与成本

3. ✏️ 编辑食谱

修改食谱信息或配方参数

调整原料比例或更换原料

修改工艺流程参数（煮面时间、温度等）

更改出餐规格或价格

可批量同步到指定售卖机

4. ⚙️ 食谱参数管理

可设置配方参数模板（如“标准口味”、“加辣口味”）

可设置出餐时间、加热功率、搅拌次数等设备指令参数

支持为不同售卖机型号调整参数

5. 📈 销售与库存关联

食谱与库存原料关联，系统可根据销售量计算原料消耗

可设置库存预警，提醒补充原料

支持销售数据统计：销量、收益、热门食谱排行

6. 🔄 食谱生命周期管理

草稿 → 启用 → 停用

支持版本管理，保留历史配方以便回滚

支持批量上架或下架

三、权限与角色
角色 权限说明
商户管理员 查看、添加、编辑、删除食谱，批量同步到售卖机
区域管理员 查看负责区域售卖机的食谱信息，不可修改
技术支持 可查看食谱参数和设备指令配置，不可修改
财务人员 可查看食谱价格和销售数据，不可修改配方
四、使用示例

例：商户“张三便利”为拉面机添加新食谱“麻辣牛肉拉面”

登录后台 → 食谱配方管理 → 新建食谱

填写信息：名称“麻辣牛肉拉面”，上传图片

配置原料：面条 150g、牛肉 50g、辣椒粉 5g、蔬菜 30g

设置制作流程：煮面 180 秒、加汤 200ml、搅拌 3 次

设置价格 ¥18，选择启用状态

保存 → 系统同步至指定拉面机 → 用户可直接下单购买
