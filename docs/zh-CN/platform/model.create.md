# ️7.2 创建/编辑

允许商户自行创建售卖机类型设备类型，在创建设备时可以选择对应的设备类型。

<b style="color: red">一经创建，不可更改板卡信息</b>

![merchant](/images/2025-11-23_114941.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

## 1. 添加板卡

![merchant](/images/2025-11-23_115050.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

### 1.1 板卡驱动

| 名称               | 描述              | 最大货道数 | 是否支持 |
| ------------------ | ----------------- | ---------- | -------- |
| Wish-noodle-v1.0   | 泡面机            | 10         |          |
| W50x-v1.0          | W50x 称重板       | 32         | ✅       |
| Wish-eLock24P-v1.0 | 24 货道电磁锁     | 24         | ✅       |
| Wish-M2301-v1.4    | 鼎商驱动板 M2301  | 100        | ✅       |
| Wish-ACH370-v1.0   | 鼎商驱动板 ACH370 | 100        |          |

`Wish-noodle-v1.0`, `Wish-ACH370-v1.0` 等系列板卡需要商户单独联系开发者。

### 1.2 选择出货方式

### 1.3 勾选货道

![merchant](/images/2025-11-23_115507.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

⚠️ 货道后期可以进行修改

### 1.4 单货道数(容量)

![merchant](/images/2025-11-23_115635.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

⚠️ 这个货道最大可以摆放多少个商品

### 1.5 掉货检测 & 升降台

- 掉货检测：是否开启了光眼
- 升降台：是否有升降货梯

### 1.6 辅助货道

> 即 门锁，如 称重板 可以指定辅助货道生成对应数量的门锁。

⚠️ <b style="color: red">第一个板卡地址不可设置为辅助货道</b>

⚠️ <b style="color: lightgreen">最多可设置 7 个副柜</b>，设置时请以生产的实际样机为准.
