# 14. 出货日志

:::tip
统一 Metadata：`appid`: `****`, 其中 `appid` 值为商户`appid`。
:::

## 请求示例

```json
{
  "current_page": "1",
  "per_page": "20"
}
```

## 参数说明

| 字段名          | 默认值 | 是否必须 | 描述            |
|--------------|-----|------|---------------|
| current_page | -   | -    | 当前页码, 默认: 1   |
| per_page     | -   | -    | 每页条数, 最大值: 20 |

## 返回结果

```json
{
  "data": [],
  "total": "15",
  "current_page": "1",
  "per_page": "20"
}
```

## 字段信息

| 字段名 | 默认值 | 描述 |
|-----|-----|----|
| id  | 1   | 主键 |
