# 1.1 Registration / Login

The official website provides multiple login methods, as follows:[https://dash.awish.vip](https://dash.awish.vip/dashboard/login?redirect=/dashboard/profile)

![login](/images/2025-11-24_150415.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

| Platform | Description                                            | Registration                               | Login |
| -------- | ------------------------------------------------------ | ------------------------------------------ | ----- |
| WeChat   | Users with WeChat app                                  | ✅                                         | ✅    |
| Google   | Web users can log in via Google authorization          | ✅                                         | ✅    |
| Password | Using email & password, users can change it themselves | ❌ Automatically generated by the platform | ✅    |

## A. WeChat QR Code (Dynamic Code)

![qrcode](/images/qrcode.jpeg){style="background: #fff; padding: 1rem; border-radius: 0.5rem; filter: blur(8px);"}

> After the QR code is displayed, open WeChat on your phone and use [Scan]. The system will automatically parse the scanned result. If login is successful, it will redirect to the management page automatically.

## B. Google Authorization

Direct link: <a title='' href="https://dash.awish.vip/auth/login?app=gg&name=google" target="_blank">Google Authorization Page</a>

> Clicking Google Authorization will automatically redirect to the Google third-party authorization page. After agreeing, the system parses the authorization data. Unregistered users will be automatically registered and successfully logged in.

## C. Email & Password

![login-form](/images/2025-11-24_150453.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

> Log in using the email address and password provided by the platform.
