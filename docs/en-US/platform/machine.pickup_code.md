# 8.3.6 Pickup Code

![2025-11-23_153307.jpg](/images/2025-11-24_171927.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

The generated pickup code is valid for 2 hours (UTC time).

## 1. Generate Pickup Code

![2025-11-24_092335.jpg](/images/2025-11-24_172013.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

Select the corresponding channel to generate a pickup code.

![2025-11-24_092335.jpg](/images/2025-11-24_172044.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

> Quantity: Optional 1–3, representing the number of products dropped per single channel.

⚠️ Devices without cart functionality (with optical sensor detection) can only set 1 code. Each channel can have only one pickup code. Newly added codes will be synced to the app device.

## 2. Delete Pickup Code

![2025-11-24_092241.jpg](/images/2025-11-24_172109.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

⚠️ Unused pickup codes will automatically expire. Deleted codes will be synced to the app device.
