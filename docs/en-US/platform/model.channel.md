## 1. Channel Details

![merchant](/images/2025-11-24_164138.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

- Channel Name: e.g., A1
- Layer Number: 0, starting from 0
- Column Number: 0, starting from 0
- Channel Capacity: 7, the maximum number of products a single channel can hold
- Motor Number: 0, starting from 0, maximum 99

## 2. Create New Channel

Channels with white buttons are unassigned and can be created as new channels.

![merchant](/images/2025-11-24_164202.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

## 3. Channel Capacity

Modifying the channel capacity will automatically update the capacity of the associated product.

## 4. Delete Channel

![merchant](/images/2025-11-24_164224.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

Deleting a channel will automatically remove all products associated with it.
