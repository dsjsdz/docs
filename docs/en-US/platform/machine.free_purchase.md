# 8.3.7 Free Claim

![2025-11-23_153307.jpg](/images/2025-11-24_172503.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

⚠️ When “Free Claim” is enabled, the device allows products to be obtained without payment.

## 1. Purchase Quantity per Person

The number of products each person can claim at one time. Default is 1, maximum is 3.

⚠️ Devices without cart functionality (with optical sensor detection) can only set 1.
