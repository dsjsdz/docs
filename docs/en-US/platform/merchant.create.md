# ➕ 2.1 Create / Join a Merchant

![merchant-create](/images/2025-11-24_152209.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

If no merchant is bound yet, the system will default to the merchant onboarding guide page. Click the "Create Merchant" button to start the merchant creation process, or click "Join Merchant" to join an existing merchant.

## 2.1.1 Create Merchant

Creating a merchant allows you to perform “Product Listing / Delisting” operations on vending machines.

![merchant-form](/images/2025-11-24_152253.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

⚠️ The platform periodically reviews merchant information to prevent abnormal accounts or invalid merchants from operating in the system.

| Field Name  | Required | Description     |
| ----------- | -------- | --------------- |
| Name        | ✅       | Merchant name   |
| Description | ✅       | Merchant intro  |
| Logo        | -        | Merchant avatar |

> After submitting the form, it must wait for approval.
> Once approved, the merchant will be automatically activated, and the user’s role, menus, and permissions will be updated accordingly.

## 2.1.2 Join Merchant

![merchant-invite](/images/2025-11-24_152326.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

> [Invitation code](/en-US/platform/merchant.invite) is valid for 12 hours, please use it promptly.

The invitation code is linked to `Merchant Information`. After use, you can choose to bind to the merchant and will automatically be added to the corresponding merchant member list.

![merchant-invite-code](/images/2025-11-24_152401.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}
