# 1.1 Language Switching

## A. Switching on the Login Page

![language](/images/2025-11-24_145024.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

> When not logged in, changing the language will automatically switch the interface language.

## B. Switching After Login

![language](/images/language-b.png){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

> When logged in, you can switch the language by clicking the header area on the management page.

## C. 支持的语言

| Language     | Supported Version      |
| -------- | ------------- |
| Simplified Chinese | v1.2.0 and above |
| Traditional Chinese | v1.2.0 and above |
| English     | v1.2.0 and above |
| Spanish | v1.2.0 and above |
| Portuguese | v1.2.4 and above |

> Other languages are currently not supported. If you need support for additional languages, please contact the <a href="mailto:cleveng@gmail.com">developer</a> .
