# 2.6 Alarm Configuration

By configuring alarm settings, you can enable real-time alarm notifications.

![2025-11-23_085533.jpg](/images/2025-11-24_155235.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

⚠️ Some alarm-related features may incur additional costs, which must be borne by the merchant. If you have any questions, please contact the merchant administrator.

| Platform | Supported     | Supported Events   | Free |
| -------- | ------------- | ------------------ | ---- |
| SMS      | ✅            | All defined events |      |
| WeChat   | ✅            | All defined events | ✅   |
| Email    | ✅            | All defined events | ✅   |
| WhatsApp | Not supported | All defined events | ✅   |
| Discord  | Not supported | All defined events | ✅   |
| DingTalk | Not supported | All defined events | ✅   |
| WeCom    | Not supported | All defined events | ✅   |
| Feishu   | Not supported | All defined events | ✅   |
| QQ       | Not supported | All defined events | ✅   |
