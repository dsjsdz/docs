# ➕ 2.2 Invitation Code

Invitation codes can only be created by the merchant administrator, and only one code can be generated within a 12-hour period. After the code expires, a new one can be created.

![merchant-invite-b](/images/2025-11-24_153937.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

Click the Invitation Code button to obtain the code.

![merchant-invite-c](/images/2025-11-24_152043.jpg){style="background: #fff; padding: 1rem; border-radius: 0.5rem;"}

⚠️ The invitation code is valid for 12 hours. Please use it promptly.
